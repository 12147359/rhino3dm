NurbsSurface¶

class rhino3dm.NurbsSurface¶

Bases: rhino3dm.Surface

Represents a Non Uniform Rational B-Splines (NURBS) surface.

static CreateRuledSurface(curveA, curveB)¶

Constructs a ruled surface between two curves. Curves must share the same knot-vector.

Args:
curveA : First curve. curveB : Second curve.
Returns:
A ruled surface on success or null on failure.
IncreaseDegreeU(desiredDegree)¶

Increase the degree of this surface in U direction.

Args:
desiredDegree : The desired degree. desiredDegree : Degrees should be number between and including 1 and 11.
Returns:
true on success, false on failure.
IncreaseDegreeV(desiredDegree)¶

Increase the degree of this surface in V direction.

Args:
desiredDegree : The desired degree. desiredDegree : Degrees should be number between and including 1 and 11.
Returns:
true on success, false on failure.
IsRational¶

Gets a value indicating whether or not the nurbs surface is rational.

MakeNonRational()¶

Makes this surface non-rational.

Returns:
true if the operation succeeded; otherwise, false.
MakeRational()¶

Makes this surface rational.

Returns:
true if the operation succeeded; otherwise, false.
OrderU¶

Gets the order in the U direction.

OrderV¶

Gets the order in the V direction.
