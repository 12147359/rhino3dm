File3dmLayerTable¶

class rhino3dm.File3dmLayerTable¶

Bases: object

Provides access to layers in the 3dm file.

FindIndex(index)¶

Retrieves a Layer object based on Index. This seach type of search is discouraged. We are moving towards using only IDs for all tables.

Args:
index : The index to search for.
Returns:
A Layer object, or null if none was found.
FindName(name, parentId)¶

Finds a Layer given its name.

Args:
name : The name of the Layer to be searched. parentId : The id of the parent Layer to be searched.
Returns:
A Layer, or null on error.
